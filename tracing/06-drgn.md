[DRGN][1]是一个相对较新的查看内核状态的工具。它是Meta开发的，作为crash的一个替代工具。

# 安装

参考最新的[安装文档][2]。

有几种安装的方式，感觉通过pip来安装，方式比较统一。

```
sudo pip3 install drgn
```

如果能运行下面的命令，看上去应该是安装好了。

```
python3 -m drgn --help
```
